# 🌿 Digital Detox App

A productivity-focused React application that helps users improve focus and reduce digital distractions using a Pomodoro-style timer with cloud-based session tracking.

---

## 🧠 Problem Statement

Modern users face constant digital distractions that reduce productivity and focus.

This application addresses this by:
- Helping users stay focused using a timed work session system
- Tracking productivity sessions over time
- Providing visual history of focus behavior
- Encouraging better time management habits

---

## 🎯 Features

- 🔐 Firebase Authentication (Signup/Login)
- ⏱️ Focus Timer (Pomodoro-based system)
- 📊 Session tracking with Firestore
- 👤 User-specific data storage
- 🚪 Protected Routes using Context API
- 📈 Total focus time calculation
- 💾 Persistent backend storage

---

## 🛠️ Tech Stack

React.js • React Router • Firebase Auth • Firestore • Context API • Tailwind CSS

---

## 🏗️ Architecture Highlights

- Context API for global authentication state
- Protected routing for secure dashboard access
- Firestore integration for real-time session storage
- Component-based modular structure
- Optimized state handling using React hooks

---

## 🔥 Key Functional Flow

- User authentication via Firebase
- Redirect to protected dashboard after login
- Start/stop focus timer sessions
- Store completed sessions in Firestore
- Retrieve and display session history per user
